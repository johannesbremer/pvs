# PVS

Initial Confect-first scaffold for the KBV-aligned PVS described in the repo specification.

Current scope:

- complete `confect/schema.ts` database assembly
- grouped table definitions in `confect/tables/*`
- shared Effect Schema healthcare primitives
- module registry placeholders for public/internal Confect surfaces
- oracle plugin skeleton and test harness placeholders

What is intentionally not implemented yet:

- runtime Confect functions
- generated refs
- FHIR/XML/xDT/print codecs
- executable oracle wrappers
- end-to-end domain workflows
